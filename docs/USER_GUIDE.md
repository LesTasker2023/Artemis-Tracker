## ğŸš€ Getting Started

### Finding Your Chat Log

Artemis reads your Entropia Universe chat log to track your hunting stats. The first time you open Artemis:

1. Click **"Select Log File"** or let Artemis find it automatically
2. Navigate to your Entropia folder (usually `C:\Program Files\Entropia Universe\`)
3. Find the `chat.log` file
4. Select it and click Open

Artemis will remember this location for future sessions.

### The Main Window

The main window has several tabs at the top:

- **Dashboard** - Your live hunting stats
- **Loot** - Everything you've looted
- **Market** - Markup library for tracking real values
- **Loadouts** - Manage your equipment setups
- **Sessions** - View past hunts

## ğŸ“Š Understanding Your Stats

### Key Stats Explained

| Stat            | What It Means                                     |
| --------------- | ------------------------------------------------- |
| **Net Profit**  | Loot value minus all costs (the real number!)     |
| **Return Rate** | Percentage of costs recovered (100% = break-even) |
| **Loot Value**  | Total TT value of everything looted               |
| **Total Spend** | Weapon decay + ammo + other costs                 |
| **Kill Count**  | Number of creatures killed                        |
| **Hit Rate**    | Percentage of shots that hit                      |
| **DPP**         | Damage Per PED spent                              |

### Reading Your Profit

- **Green numbers** = You're making money! ğŸ’š
- **Red numbers** = You're losing money ğŸ’”
- **Yellow numbers** = Close to break-even

## ğŸ–¥ï¸ The Popout Window

The popout is a small overlay you can keep on top while playing.

### Opening the Popout

1. Click the **popout icon** in the top-right corner
2. A small window appears with your live stats
3. Drag it anywhere on screen
4. It stays on top of Entropia!

### Choosing a Layout

Click the **layout button** to switch between:

- **Mini** - Tiny bar with essential stats
- **Horizontal** - Wide bar across your screen
- **Vertical** - Tall sidebar
- **Grid** - Card layout with multiple stats

### Customizing Your Stats

1. Click the **settings/edit button** âš™ï¸
2. Drag stats to reorder them
3. Click **+ Add** to add more stats
4. Click **X** on any stat to remove it
5. Use **presets** for quick layouts:
   - Economy (profit focused)
   - Efficiency (performance focused)
   - Skills (skill gain focused)

### Additional Expenses

Track costs that Artemis can't detect automatically:

1. Open settings in the popout
2. Enter costs for:
   - **Armor** - Extra armor decay/repairs
   - **FAP** - Healing tool costs
   - **Misc** - Pills, probes, etc.
3. Toggle **AE** to include/exclude these from profit

### MU Toggle

- Click **MU** to switch between TT values and markup values
- Shows your real profit based on market prices

## âš”ï¸ Loadouts

Loadouts let you save your equipment setups for accurate cost tracking.

### Creating a Loadout

1. Go to the **Loadouts** tab
2. Click **+ New Loadout**
3. Name your loadout (e.g., "Mulciber Hunting")
4. Add your equipment:
   - **Weapon** - Search and select your gun
   - **Amp** - If using an amplifier
   - **Scope/Sight** - If equipped
   - **Armor Set** - Your armor pieces
   - **Plates** - Any armor plates

### Setting Active Loadout

- Click a loadout to make it active
- Artemis uses this loadout's costs for calculations
- Switch loadouts anytime mid-hunt

## ğŸ’° Markup Library

Track real market values to see your true profit.

### Adding Markup Values

1. Go to the **Market** tab
2. Find an item using search
3. Set markup as:
   - **Percentage** (e.g., 115% = 15% above TT)
   - **Fixed PED** (e.g., 2.50 PED per item)
4. Click **Save**

### Quick Access Tabs

- **All** - Every item in database
- **Favorites** - Items you've starred â­
- **Custom** - Items with custom markup set
- **Recent** - Items from your current session
- **Ignored** - Items you've hidden

### Ignoring Items

Don't want nanocubes messing up your stats?

1. Find the item
2. Click the **Ignore** button (eye icon)
3. Item won't count in your totals
4. Find it in the Ignored tab to restore

**Q: Why is my profit wrong?**

- Make sure ammo costs are accurate

- Restart Artemis

- Switch log files when changing characters
